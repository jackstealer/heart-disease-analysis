================================================================================
🔌 API INTEGRATION
================================================================================

The frontend connects to Flask backend via Vite proxy:

PROXY CONFIGURATION (vite.config.js):
-------------------------------------
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:5000',
      changeOrigin: true,
    }
  }
}

API ENDPOINTS USED:
------------------
POST /api/predict          - Submit assessment
GET  /api/model-info       - Get model info
GET  /api/stats            - Get statistics
GET  /api/health           - Health check

EXAMPLE API CALL:
----------------
import axios from 'axios'

const data = {
  BMI: 28.5,
  Smoking: 'No',
  // ... other fields
}

const response = await axios.post('/api/predict', data)
console.log(response.data)

🎨 CUSTOMIZATION GUIDE
================================================================================

1. CHANGE COLORS
   -------------
   Edit: frontend/tailwind.config.js
   
   colors: {
     primary: {
       600: '#YOUR_COLOR',
     }
   }

2. MODIFY 3D HEART
   ---------------
   Edit: frontend/src/components/Heart3D.jsx
   
   <MeshDistortMaterial
     color="#YOUR_COLOR"
     distort={0.3}
     speed={2}
   />

3. UPDATE BRANDING
   ---------------
   - Logo: Edit Navbar.jsx
   - Name: Change "CardioAI" throughout
   - Favicon: Replace in public/

4. ADD NEW PAGE
   ------------
   1. Create: src/pages/NewPage.jsx
   2. Add route in App.jsx:
      <Route path="/new" element={<NewPage />} />
   3. Add to Navbar.jsx navigation

🐛 TROUBLESHOOTING
================================================================================

ISSUE: Port 3000 already in use
SOLUTION: Change port in vite.config.js or kill process

ISSUE: 3D animation not showing
SOLUTION: Check WebGL support, try different browser

ISSUE: API connection errors
SOLUTION: Ensure Flask backend running on port 5000

ISSUE: npm install fails
SOLUTION: Delete node_modules, run npm cache clean --force

ISSUE: Build errors
SOLUTION: Check Node.js version (need 16+)
