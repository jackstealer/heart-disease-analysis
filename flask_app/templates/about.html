About This Project

Project Overview

Heart disease remains one of the leading causes of mortality worldwide. This project leverages Tableau as a powerful data visualization and business intelligence tool to analyze heart disease data and transform raw data into meaningful dashboards.

Technical Architecture

Our solution follows a comprehensive data pipeline:

  1. Data Collection: Heart disease dataset from UCI ML Repository
  2. Data Storage: MySQL database for structured storage
  3. Data Processing: SQL operations and Python scripts for data preparation
  4. Visualization: Tableau dashboards and stories
  5. Web Integration: Flask application for dashboard embedding

Key Features

  • Interactive dashboards with multiple visualizations
  • Data filters for demographic analysis
  • Risk factor identification and correlation analysis
  • Story-based narrative for data insights
  • Responsive design for multiple devices
  • Real-time data updates

Technologies Used

  • Tableau Desktop & Server
  • Python 3.8+
  • Flask Web Framework
  • MySQL Database
  • Pandas & NumPy
  • Bootstrap 5
