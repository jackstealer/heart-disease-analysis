================================================================================
HOW IT WORKS (NO HARDCODING!)
================================================================================

1. DATA ANALYSIS PHASE (train_transformer.py)
   - Loads Heart_new2.csv
   - Calculates statistics for each feature:
     * Numeric: min, max, mean, std, median, q25, q75
     * Categorical: unique values, value counts, most common
   - Calculates feature importance (correlation with target)
   - Saves all to transformer_metadata.json

2. PREDICTION PHASE (transformers_model.py)
   - Loads metadata from JSON
   - For each patient:
     * Compares values against data-driven thresholds
     * Weights by feature importance
     * Calculates risk score
     * Returns probability and risk level

3. EXAMPLE: BMI THRESHOLD
   - NOT hardcoded as "BMI > 30"
   - INSTEAD: Compares against actual data quartiles
   - If BMI > 75th percentile from data → risk +2
   - If BMI > median from data → risk +1

4. EXAMPLE: AGE RISK
   - NOT hardcoded age ranges
   - INSTEAD: Uses actual AgeCategory distribution
   - Weights by correlation with heart disease
